The following projects are relevant to GPE:

Base software/libraries:

  • uSQLd Database Engine A minimimalist database engine base upon SQLite
  • GTK+ Toolkit Graphical User Interface (GUI) toolkit for X11 (not maintained by the GPE project)
Applications:
  • Application Manager, the so to say main menu of the PDA user interface. It launches applications and ensures that certain applications are only launched once.
  • System setup, which will consist of several smaller parts to setup the PDA system, for example date and time config, network configuration, multi-user configuration (this is new for PDAs!)
  • Address book.
  • Calendar.
  • Notebook.
  • Todo list.
  • Alarm clock, for me one of the most important applications.
  • Media Player, a nice add-on feauture, good for boring meetings and to impress your friends.
  • Daily summary
  • Time tracker
  • Voice memo recorder
  • Games ;)
  • Mail Client
  • ... and for sure many more to come
Utilities:
  • Multi-user login
  • Bluetooth PIN manager
  • Modal help system
Libraries:
  • libgpewidget

See the Wiki page CurrentThingsToDo for a list of things that particularly need attention.
