Function GPE-SCAP is a tool to create screenshots on arbitary devices running GPE. it is able to save screenshots on the device and to upload these to http://scap.linuxtogo.org. .png" border="0" alt="Screenshot">
Coordinator(s) The coordinator for this sub-project is Florian Boor (florian.boor@kernelconcepts.de) . If you want to contribute to this project, please send a mail to the GPE mailinglist or contact him directly.
Status

The source code can be found in SVN, module base/.

Distributions known to ship are:

  • OpenZaurus
  • Angstrom

is distributed under the GPL.

Technical

GPE-SCAP has a nicely designed GUI using libglade and should work in any X based environment. The screenshot upload feature is implemented using libsoup.

Project History
  • 2005 Project started as gpe-screenshot by Rene Wagner
  • 2006 Forked with a new name, extended and adapted to support scap.linuxtogo.org by Florian Boor.
