Function GPE-Edit is a simple text editor for GPE. It supports editing single text files and has functions for text search and replace. It supports UTF8 encoded files and is the first application supporting the GPE help interface.
Coordinator(s) The coordinators for this sub-project are Philip Blundell (philb@gnu.org) and Florian Boor (florian@linuxtogo.org). If you want to contribute to this project, please contact the coordinators or the GPE mailinglist.
Status

GPE-Edit is stable for production use and actively maintained.

The source code can be found in SVN, module base/gpe-edit.
