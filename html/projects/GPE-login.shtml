Coordinator

The coordinator for this sub-project is Philip Blundell (philb@gnu.org) . If you want to contribute to this project, please contact him directly.

Function

The login program serves two purposes. First, it provides password protection for your personal data; second, it allows multiple users to share a single devices.

Status

The source code can be found in CVS, module gpe-login. Precompiled binaries for iPAQ are available in the Familiar unstable feed.
