Function

The login program serves two purposes. First, it provides password protection for your personal data; second, it allows multiple users to share a single device.

gpe-dm starts the X server, then runs /etc/X11/Xinit as root (which in turn calls in everything in /etc/X11/Xinit.d). Once Xinit has finished, it launches gpe-login, which runs /etc/X11/gpe-login.setup (also as root), then displays its UI and prompts the user to log in. Once the user is authenticated, it runs ~/.xsession if it exists, and /etc/X11/Xsession if not, both as the user.

The Xsession file needs to end by running the window manager, session manager, or some other long-living thing. Once Xsession returns, gpe-dm will do a server reset and start the whole cycle again.

Processes started by Xinit survive into the user's session, whereas things run from gpe-login.setup are terminated before Xsession starts.
