Function

GPE-Conf is a set of configuration tools to perfor the basic configuration tasks on a mobile device. It is designed to expose all necessary settings in an easy to use and low intrusive way. It is even able to set up some tasks that are subject of more advanced use of a device such as serial port usage and multi user setups.

The GPE-Conf package contains another small tool - GPE-Info to provide information about the device and it current status as well.

Coordinator(s) The coordinator for this sub-project is Florian Boor (florian@linuxtogo.org). If you want to contribute to this project, please contact him or the GPE mailinglist.
Status

GPE-Conf is stable for production use and actively maintained and developed. Areas that want improvement include UI usability and a proper backend design.

The source code can be found in SVN, module base/gpe-conf.

Distributions known to ship are:

  • Familiar
  • OpenZaurus
  • Angstrom

is distributed under the GPL.

It currently includes the following applets:

  • Date and time
  • Display setings like brightness and calibration
  • GUI style and theme selection
  • Power management / auto sleep
  • Network settings
  • Button mapping, virtual- and external keyboards
  • Login settings
  • Owner information
  • PCMCIA/CF card information and driver assignment (deprecated)
  • Serial port assignment
  • Sound settings
  • USB port settings
  • User manager

The GPE-Conf frontends usually run width normal user provileges while tasks that require root permissions are performed by a separate process. It allows privileged users to make all settings exposed by GPE-Conf available to all users. It uses an IPC communication mechanism for the communication between frontends and backend.
