Function

The GPE user interface is heavily icon-oriented, and it's not always completely obvious what the exact effect of each button is going to be. On a desktop system this would be dealt with by "mouse-over" tooltips, but these don't really work when your pointing device is a stylus.

GPE-what implements a modal help system for the GPE environment. It comes in two parts, a small application that lives in the window manager dock, and some client-side library code which is resident in libgpewidget. Clicking on the button in the dock causes all applications to enter "help mode"; their normal UI is suspended, and the next button click is instead interpreted as a request for help on the selected widget.

Status

The source code can be found in CVS, module gpe-what. Precompiled binaries for ARM PDAs are available in the Familiar unstable feed.
