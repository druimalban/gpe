Coordinator

The coordinator for this sub-project is Philip Blundell (philb@gnu.org). If you want to contribute to this project, please contact him directly.

Function

This is very obvious. It stores contact details such as addresses and is able to categorize them. GPE-Contacts uses DBUS to connect to service provider applications (GPE-Beam and GPE-Bluetooth) to provide IrDA (IrOBEX) and Bluetooth (BtOBEX) support for beaming them in vCard format.

Status

GPE-Contacts is in CVS, actively maintained, in Familiar 0.8.2 release and the unstable feed.
