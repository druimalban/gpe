Function gpe-clock is an alarm clock dock application. It displays the time in the system tray and manages simple alarms (single or weekly). It can be configured to display a digital or an analogue clock face. .png" border="0" alt="Screenshot">
Coordinator(s) The coordinator for this sub-project is Philip Blundell (philb@gnu.org). If you want to contribute to this project, please contact the coordinator or the GPE mailinglist.
Status

The application is considered to be stable.

The source code can be found in SVN, module base/.

Distributions known to ship are:

  • Familiar
  • OpenZaurus
  • Angstrom

is distributed under the GPL.

Technical

gpe-clock relies on atd for alarm scheduling. The menu option to set the time makes use of gpe-conf.
