Function

This program allows short voice messages to be recorded and played back through the PDA's internal audio system. The libgsm codec library can be used to compress the data for storage.

I'm not sure what the user interface should be like. Obviously, on the iPAQ, the record button should start and stop recording. But there also needs to be some means to choose where the files are saved and to play them back.
