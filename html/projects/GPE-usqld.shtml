uSQLd - a micro-SQL server

uSQLd is basically a network RPC wrapper for D. Richard Hipp's fantastic open source sql library SQLite: http://www.hwaci.com/sw/sqlite/

The first unstable release of GPE is in handhelds.org FTP: ftp://handhelds.org/pub/projects/gpe/unstable/SOURCES/usqld-0.2.1.tar.gz

Note: uslqd requires any sqlite 2.x to be compiled with -DTHREADSAFE=1
