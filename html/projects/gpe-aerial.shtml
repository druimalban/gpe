Function GPE-Aerial is a too to search wireless lans and to connect to these. It is designed for devices with old or bad wifi drivers that do not support passive scanning like the drivers used in many Familiar Linux releases. .png" border="0" alt="Screenshot">

Technical

GPE-Aerial makes use of the Prismstumbler wireless LAN scanner engine which is able to find all sorts of wireless LANs if it is able to work with the driver of your hardware. For more modern devices with recent Linux 2.6 its backend should be replaced with something using passive scanning only.
