Subprojects:

Function gpe-timesheet is used to keep a list of tasks and track the time spent on them. These taks can contain sub-tasks and as an option gpe-timesheet allows to use the tasks from gpe-todo as well. It provides a journal view displaying the tracked data and summarizing times. Sime some other PIM application gpe-timesheet supports the Maemo software environment. .png" border="0" alt="Screenshot">
Coordinator(s) The coordinators for this sub-project are Philip Blundell (philb@gnu.org) and Philippe De Swert (philippedeswert@scarlet.be). If you want to contribute to this project, please contact the coordinators or the GPE mailinglist.
Status

The application should be stable and working but lacks some intensive testing.

The source code can be found in SVN, module base/.

Distributions known to ship are:

  • Familiar
  • OpenZaurus
  • Angstrom

is distributed under the GPL.

Technical

Like the other PIM applications gpe-timesheet makes use of a SQLite database to store data but it does not have a library for abstracting the data backend. Latest versions do not call an external application to display the journal anymore.
