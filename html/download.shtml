Current unstable GPE binaries for iPAQ-familiar (in .ipk format) can be downloaded form handhelds.org ftp : ftp://ftp.handhelds.org/pub/projects/gpe/

At present these are unstable and active development sources can be found in the GPE CVS archive

GPE has made it into Familiar!

So if you install the X11 branch of Familiar V0.6-pre1 (or later) you will get the latest almost stable GPE packages.
