One of the goals of the GPE project is encouraging developers for our platform to conform to a number of standards separate from the core GPE software.

SQL schemas

The core storage model for environment-wide data such as PIM data and user prefferences is that of a relational database, implemented through the ubiquitous SQL (Structured Query Language).
